# Prism User Docs

This section is for user-facing Prism documentation. It is organized with the
Diataxis model:

- **Tutorials** teach a complete first-time path.
- **How-to guides** solve a specific task.
- **Reference** documents facts, fields, labels, and options.
- **Concepts** explain how Prism works and why it is shaped this way.

The site service renders this section at `/docs`.
