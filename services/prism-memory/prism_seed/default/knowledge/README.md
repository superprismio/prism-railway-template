# Prism Knowledge Scaffold

This directory is the evergreen knowledge area for the starter.

The expected structure is:

- `kb/docs/` for canonical markdown knowledge docs
- `kb/metadata/` for sidecar metadata JSON
- `kb/indexes/` for generated retrieval indexes
- `kb/activity/` for knowledge activity log
- `kb/state/` for knowledge pipeline state
- `kb/triage/inbox/` for incoming promotion candidates

The starter leaves these mostly empty on purpose.
